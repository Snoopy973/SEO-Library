       SEO LIBRARY — Guide d'installation
═══════════════════════════════════════════════════

INSTALLATION SUR UN NOUVEAU MAC :

  1. Copie le dossier "SEO-Library" sur le Bureau
  2. Double-clic sur "INSTALLER.command"
  3. Attends que l'installation se termine
  4. L'app "SEO Library" apparaît dans ton Dock !

UTILISATION :

  → Clique sur "SEO Library" dans le Dock
  → Choisis l'outil à lancer dans le menu
  → C'est tout !

OUTILS DISPONIBLES :

  🔍 Dashboard SEO (Streamlit)
     - Ré-optimisation de contenus
     - Title Optimizer
     - SERP Analyzer
     - Scraper Produits Shopify/WooCommerce
     - Positions & Opportunités (filtre par matière)

  🛒 Scraper Shopify/WooCommerce
     - Entre une URL → récupère tous les produits
     - Analyse par matière, couleur, coupe
     - Export Excel automatique

AJOUTER UN NOUVEL OUTIL :

  1. Crée ton script dans tools/mon-outil/
  2. Ajoute une entrée dans config/apps.json
  3. Relance l'INSTALLER.command
  4. C'est dispo dans le menu !

PRE-REQUIS :

  - macOS 12+
  - Python 3 (vérifie avec : python3 --version)
  - Les dépendances s'installent automatiquement
